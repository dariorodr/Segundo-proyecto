# Segundo-proyecto - Gestor de Turnos de canchas de futbol.

Sistema web hecho para administrar y registrar turnos de reserva de canchas de futbol.

Tecnologias utilizadas

-Bootstrap
-HTML/CSS
-Javascript (alerta aevlist)
-Mysql
-Python/Flask

Como ejecutarlo:

## Cómo ejecutarlo
1. Cloná el repositorio
2. Instalá las dependencias: `pip install -r requirements.txt`
3. Activar el enterno virtual: venv/Scripts/activate
4. Ejecutá `py app/app.py`
5. Abrí el navegador en `http://localhost:5000`
