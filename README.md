# Segundo-proyecto - Gestor de Turnos de canchas de futbol.

App web hecha para administrar y registrar turnos de reserva de canchas de futbol.

# Funcionalidades destacadas

- Gestión completa de turnos: creación, edición y eliminación.
- Administración de canchas: alta, baja y clasificación por nombre, tipo, capacidad.
- Administración de precios: alta, baja y clasificación por nombre y tipo de precio (con/sin iluminación).
- Filtros de búsqueda de turnos por cancha, fecha, cliente, hora, tipo de césped, duración y capacidad.
- Panel de recaudación con filtros por cancha,tipo de cancha y fecha.
- Cálculo automático del monto total recaudado según los turnos registrados.

# Tecnologias utilizadas

-Python/Flask
-Mysql
-Bootstrap
-HTML/CSS
-Javascript (alerta aevlist)


# Cómo ejecutarlo:

1. Clonar el repositorio.
2. Instalar las dependencias: `pip install -r requirements.txt`.
3. Activar el entorno virtual: venv/Scripts/activate.
4. Ejecutar `py app/app.py`.
5. Abrir el navegador en `http://localhost:5000`.
